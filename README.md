# GDP-Analyzer
Python-based system to analyze World Bank GDP data and turn it into meaningful insights
functional programming principles and enforcing the Single Responsibility Principle (SRP) applied
The key features include:
- Building modular components for clean architecture
-Configuration-driven filtering & statistical operations
-Multiple chart types for GDP insights
-Robust error handling for configs & datasets
This project shows how data‑driven insights can support real‑world economic analysis and decision‑making. It connects software design principles with practical data analytics, resulting in a scalable and professional dashboard.
